# Procédure dd'import de géométries dans une opération

Cette fiche décrit le workflow d'import d'un fichier geojson

## Objectif
Permettre à un utilisateur d'assigner une géométrie (point - ligne - polygone) dans une opération.

## Workflow
Vous avez une forme à assigner dans ATLAS.

1. **Dans QGIS, selectionnez votre entité à exporter**

2. **Clic droit sur la couche**

3. **Exporter les entitées selectionnées**

4. **Prennez le type GEOJSON et sauvegardez le fichier**

5. **Charger le dans ATLAS**

6. **Passez à autre chose. 👍**
